Html: A necessary prerequisite for every website.

"What is HTML?"

HTML stands for Hyper Text Markup Language.
It is the combination of Hypertext and Markup language.
The term "Hypertext" defines the link between the web pages and markup language.
HTML is the standard markup language for creating Web pages.
It describes the structure of a web page.
It consists of a series of elements. These elements tell the browser how to display the content.
These elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.
Uses
HTML is used to create the structure of web pages that are displayed on the World Wide Web (www). It contains "Tags" and "Attributes" that are used to design the web pages. Also, we can link multiple pages using Hyperlinks.

Html Semantics

Semantic HTML refers to syntax that makes the HTML more comprehensible by better defining the different sections and layout of web pages. It makes web pages more informative and adaptable, allowing browsers and search engines to better interpret content.
Html Semantics clearly describes its meaning to both the browser and the developer. Examples of semantic elements: 'form', 'table', and 'article' - Clearly defines its content.
Examples of non-semantic elements: 'div' and 'span" - Tells nothing about its content.
Other examples of Html Semantics include: 'article', 'footer', 'header', etc.

HTML Tags
Tags are simple pieces of data — usually no more than one to three words — that describe information on a document, web page, or another digital file. Tags provide details about an item and make it easy to locate related items that have the same tag.

Container Tags

Container tags are generally divided into three parts, i.e., opening tag, content(which will display on the browser), and closing tag. In the content part, they can also contain some other tags. Some commonly used container tags are:
Essential Tags
: <'html'>: This marks the beginning and ending of the webpage also it tells that the document is an HTML document.
<'head'>: This tag is used to define the head part of the document which contains the information related to the webpage.
<'title'>: This tag stores the description of the web page, whatever given in these tags appears on the tab name while opened by the browser.
Headings
(h1>/h1> to h6>/h6): It is used for including headings of different sizes ranging from 1 to 6.
Text formatters
(p,/p): When paragraphs are needed to be included, this tag is used.
(b,/b): Makes the contained text to bold.
(i,/i): Makes the contained text to italic.
Hyperlinks
(ahref=”link.com”,/a) : When we link some other webpages we add the hyper links to other webpages using this …tag.
Other container tags are:
(button…/button): This is used in many ways but mainly used to manipulate dom by adding events and many more.
(div…./div): This defines a section in a document. The webpage can be divided to different sections using the (div.../div)
(iframe src=”link.com> /iframe): When some other document is to be embedded like some video or image into HTML we use this tag.
(nav…/nav): Defines a navigation bar that contains a set of menu or a menu of hyperlinks.
(script…/script) : This contains the javascript code that adds interactivity to the webpage.
(ol>…/ol): This tag is used to create ordered lists.
(ul>…/ul): This tag is used to create unordered lists.
(li>…/li): This tag is used to add list items.
(!-- --) Comment: Anything between these tags is not displayed on the screen,useful for making notes to yourself or to others who may view the source code of the web page.
